# tic-tac-toe
A simple tic tac toe game

## User Interface Breakdown 

* Game Title
* 3x3 grid
* The grid is clickable
* The grid cells display correct player sign
* Displays a message informing the current player
* Shows us who won the game( X or O)
* Shows us if the game ended in a draw
* Restart button - will restart the entire game
